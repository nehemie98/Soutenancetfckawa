Présentation du projet de fin d'études - Excellence technologique

Plateforme Web de Gestion Locative

Solution complète révolutionnant la gestion des locations immobilières. Architecture modulaire et interface intuitive offrant une expérience fluide pour propriétaires et locataires.

Fonctionnalités principales :

  • Gestion centralisée des propriétés
  • Système de réservation en ligne
  • Paiements sécurisés
  • Contrats numériques
  • Calendrier de disponibilité
  • Tableau de bord analytique
  • Système de notation et d'avis

Technologies utilisées :

React.js, Node.js, MongoDB, API RESTful, AWS, Docker
